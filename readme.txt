Simple plugin to wrap MediaText zone tags around content and/or comments

== Description ==
We can explicitly control what parts of our sites to display MediaText's ads by using "ad zone" div tag as follows.

`<div id="mediatext">`
`...Content we want ad links appear here...`
`</div>`

This Wordpress plugin can help wrapping MediaText's ad zone tags around post content and/or comments easily.

== Installation ==

1. Upload `mediatext_ad_wrap.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Settings - MediaText Ad Wrap' menu for selecting options
